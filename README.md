# 24小时负荷曲线生成工具

一个专业的电力负荷数据分析工具，支持Excel文件上传、数据处理和可视化展示。

## 🌟 功能特性

- 📊 **智能数据识别** - 自动识别Excel中的负荷数据
- 📈 **专业图表生成** - 生成24小时负荷曲线图
- 📋 **数据预览** - 实时预览处理后的数据表格
- 📁 **批量处理** - 支持多个Excel文件批量处理
- 💾 **数据导出** - 支持导出处理后的CSV文件
- 🎨 **现代化UI** - 采用专业分析仪表板配色方案

## 🚀 在线使用

访问在线版本：[GitHub Pages链接]

## 📦 本地部署

1. 克隆仓库
```bash
git clone [仓库地址]
cd 负荷曲线生成工具
```

2. 直接打开 `负荷处理数据.html` 文件即可使用

## 🎨 技术栈

- **前端框架**: 原生HTML + Tailwind CSS
- **图表库**: Chart.js
- **数据处理**: XLSX.js
- **文件操作**: FileSaver.js + JSZip

## 📋 支持的文件格式

- Excel文件 (.xlsx, .xls)
- CSV文件 (.csv)

## 🔧 开发说明

这是一个纯前端应用，无需后端服务。所有数据处理都在浏览器本地完成，确保数据隐私安全。
